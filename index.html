Haniwa archive

Searchable database of Japanese Kofun-period terracotta figures

Hello, and welcome to the Haniwa Database. This is a searchable collection of Haniwa which currently include most of the Haniwa from Gunma museum

Future plans for the project

  • Adding more pictures to each Haniwa
  • Incorporating all the Haniwa in Japan and internationally
  • Adding map references to all the clusters of Haniwa with incorporated map on the website
  • Adding a 3D scan feature where you can view the Haniwa in 3D
  • Adding a better translation function
